# Engineering documentation

This folder is the implementation-facing engineering baseline extracted from `docs/QUILL-PRD.md` sections 10 and 11 and adapted to the current codebase state.

## Current docs

- `architecture.md` — system boundaries and package responsibilities
- `module-contracts.md` — public contracts and invariants by module
- `runtime-model.md` — threading, task execution, and cancellation model
- `data-layout.md` — persistent storage layout and file formats
- `quality-gates.md` — lint/type/test/build gates and release quality bars
- `roadmap-mapping.md` — PRD commitments mapped to current implementation status
- `diagnostics-runbook.md` — diagnostics bundle spec, redaction rules, and support triage flow
- `security-advisory-workflow.md` — private advisory intake, triage, patch, and coordinated disclosure workflow
