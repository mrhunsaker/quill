# Research Alchemist

The **Research Alchemist** is a showcase Quillin that demonstrates the power of filesystem integration and automated document synthesis.

## Features
- **Automated Siphoning**: Scans `~/quill_research` for bulleted notes.
- **Source Tracking**: Prefixes every extracted item with the filename and line number.
- **One-Click Synthesis**: Turns a directory of disjointed notes into a structured list in your editor.

## Setup
1. Create a folder named `quill_research` in your home directory.
2. Add `.txt` or `.md` files containing notes starting with `-` or `*`.
3. Run the "Siphon Knowledge" command from the Tools menu.

## Accessibility
- Announces the number of items siphoned.
- Reports errors via the system announcement bar.
- Fully keyboard-operable via the Tools menu.
