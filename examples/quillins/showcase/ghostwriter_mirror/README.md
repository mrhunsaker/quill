# Ghostwriter's Mirror

The **Ghostwriter's Mirror** is a showcase Quillin that demonstrates the power of dynamic text transformation within QUILL.

## Features
- **Persona Shifting**: Instantly change the "vibe" of your writing.
- **Thematic Mapping**: Leverages a handler-based approach to map common colloquialisms to persona-specific language.
- **Non-Destructive**: Replaces only the current selection, allowing for easy experimentation.

## How to Use
1. Select a sentence or paragraph in your document.
2. Go to the **Format** menu.
3. Choose between "Legal Formalist" (for a professional, rigid tone) or "Victorian Poet" (for a romanticized, dramatic tone).

## Accessibility
- Announcements provide immediate confirmation of the transformation.
- Operates entirely via standard menu commands.
