# Accessibility Auditor

The **Accessibility Auditor** is a showcase Quillin that helps writers create documents that are friendly to screen-reader users.

## Features
- **Heading Hierarchy Check**: Warns you if you skip heading levels.
- **Alt-Text Validation**: Flagging image markers that lack descriptions.
- **Noise Reduction**: Detecting redundant punctuation that can disrupt screen-reader flow.

## How it Works
Run the "Audit Accessibility" command from the Tools menu. The plugin scans your entire document and inserts a report detailing issues and where they are found.

## Accessibility
- Fully keyboard-accessible via the Tools menu.
- Announces the number of issues found immediately to the user.
