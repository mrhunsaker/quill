AI Writing Prompts

A bundled QUILL Quillin that contributes a curated set of AI writing and editing prompts to the Prompt Library.

Contents

prompts.json contains 7 prompts across the Writing, Editing, Structure, and Research categories:

  • Expand This (Writing)
  • Vary Sentence Rhythm (Writing)
  • Make More Vivid (Writing)
  • Write a Title (Structure)
  • Generate Outline (Structure)
  • Suggest Supporting Evidence (Research)
  • Plain Language (Editing)

Usage

These prompts are loaded automatically when you open the Prompt Library (Tools > AI Assistant > Prompt Library). They appear alongside the built-in prompts and can be used with any configured AI provider.

Sharing prompt packs

Export your prompts via File > Export .pqp in the Prompt Library dialog to share them as a .pqp (QUILL Prompt Pack) file. Others can import them via File > Import .pqp.
